# AdventureWorksDW2019  Dashboard

## Overview

This project is an interactive Power BI dashboard built using the Microsoft AdventureWorksDW2019 sample database, which contains sales data for bicycles and related accessories.

The dashboard was developed to analyze different aspects of the business, including sales performance, customers, employees, and products. It consists of four interactive report pages designed to provide meaningful business insights.

---

## Tools Used

- Power BI
- DAX
- SQL Server
- AdventureWorksDW2019

---

## Dashboard Pages

### 1. Sales Overview

This page provides an executive overview of the company's sales performance through interactive KPIs and visualizations.

Key insights include:

- Best-performing sales year
- Total sales amount
- Total order quantity
- Top country by sales
- Sales target achievement using KPI visualization
- Yearly sales trend analysis
- Dynamic comparison of sales amount, tax, freight cost, and order quantity across countries using Dynamic Measures
- Dynamic analysis by country, product category, and customer gender using Dynamic Attributes

---

### 2. Customers

This page focuses on customer analysis and purchasing behavior.

Key insights include:

- Total number of customers
- Average annual customer income
- Top customer based on total purchases
- Interactive Top N customer analysis
- Customer age group distribution
- Purchasing behavior based on order quantity and purchase value
- Annual income comparison across different age groups

The analysis shows that most customers place relatively few orders and purchase fewer items, while the overall purchase value remains relatively high. It also indicates that **Middle-aged** customers represent the largest customer group, while **Middle-aged** and **Older Adult** customers generally have higher purchasing power.

---

### 3. Employees

This page analyzes employee performance using interactive filters for year, department, and employee.

The dashboard provides insights into:

- Employees with the highest leave hours
- Hiring trends over the years
- Sales team performance
- Relationship between leave hours and sales performance
- Employee gender distribution
- Average employee age
- Best-performing sales employee
- Total sales generated by the sales team
- Sales target achievement using a Gauge visualization

These analyses help evaluate employee performance and support business decisions regarding workforce management.

---

### 4. Products

This page evaluates product performance and sales.

Key insights include:

- Best-selling product
- Best-performing product subcategory
- Best-performing product category
- Sales by product subcategory
- Order quantity by product category
- Total sales by product category
- Relationship between product sales and order quantity

These visualizations help identify the most successful and highest-demand products.
